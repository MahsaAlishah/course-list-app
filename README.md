# Course List App

A simple React application that displays courses in a modern card-based layout.

## Features

* Display course information
* Show ratings, tags, and start dates
* Dynamic status badges (Active, Completed, Upcoming)
* Reusable React components
* Responsive and clean UI

## Technologies

* React
* JavaScript (ES6+)
* CSS
* Vite

## Run Locally

```bash
npm install
npm run dev
```

## Learning Goals

This project was built to practice:

* React Components
* Props
* List Rendering
* Component Composition
* Conditional Styling
